# Code of Conduct

This project was built so kids could play with their friends. The same
spirit applies here: be kind.

- Be respectful and constructive in issues, PRs, and discussions.
- Assume good faith — many contributors (and users) are beginners; this is
  explicitly a learning-friendly project.
- No harassment, personal attacks, or discriminatory language of any kind.

Maintainers may edit, lock, or remove content and ban contributors who
don't follow these rules. To report a problem privately, contact the
maintainer via GitHub ([@mauroartizzu](https://github.com/mauroartizzu)).

This is a lightweight adaptation in the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/).
